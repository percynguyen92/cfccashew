Just a simple tool to organize my work in Nigeria

## Frontend Setup

This project uses [Tailwind CSS](https://tailwindcss.com/), [DaisyUI](https://daisyui.com/) and [Alpine.js](https://alpinejs.dev/) via Vite.

Install the JavaScript dependencies and run the development server:

```bash
npm install
npm run dev
```

Build assets for production with:

```bash
npm run build
```
